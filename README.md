# Премиальный лендинг ООО «ПАРСЕК» (v1.0)

Высокотехнологичный B2B-лендинг для оптового поставщика телематического оборудования.

## Технологический стек (2026 Edition)
- **Framework:** Next.js 15+ (App Router, React 19)
- **Styling:** Tailwind CSS 4, shadcn/ui
- **Animations:** GSAP 3 + ScrollTrigger, Framer Motion
- **Smooth Scroll:** Lenis
- **3D Graphics:** Spline (@splinetool/react-spline)
- **Icons:** Lucide React
- **Validation:** React Hook Form + Zod
- **Type Safety:** TypeScript

## Ключевые особенности
- **Cinematic Experience:** Плавный скролл и оркестрированные анимации появления блоков.
- **3D Hero:** Интерактивная сцена Spline для максимального wow-эффекта.
- **Dark/Light Mode:** Поддержка тем с плавным переключением.
- **Performance:** Оптимизированные ассеты и React Server Components.
- **SEO Ready:** JSON-LD разметка, семантические теги и мета-данные.
- **Responsive:** Идеальное отображение на всех устройствах (Mobile-first).

## Запуск проекта
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
3. Откройте [http://localhost:3000](http://localhost:3000)

## Структура данных
Все текстовые данные (отзывы, продукты, информация о компании) вынесены в JSON-файлы в директории `src/data/` для удобного редактирования.
